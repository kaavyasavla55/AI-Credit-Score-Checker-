Features

Why CreditAI Works

Built on 307,511 real applicants with a rigorous ML pipeline designed for accuracy, fairness, and explainability.

🤖
XGBoost ML Engine

1,000-tree gradient boosted model with optimised threshold tuning for maximum F1 on imbalanced classes.

🔍
SHAP Explainability

Every prediction comes with per-feature SHAP values — satisfying EU AI Act Article 13 transparency requirements.

⚖️
Fairness Audited

Disparate Impact Ratio analysis using the Four-Fifths Rule. Protected attributes excluded from all inputs.

📊
CIBIL-Scale Scoring

Output mapped to 300–900 aligned with India's CIBIL system. Scores above 650 recommended for approval.

🧮
Alternative Data Features

Credit-to-income ratio, annuity burden, loan-to-goods, employment stability — no credit file required.

⚡
Instant Assessment

Applicant features are scaled and passed through the trained model in milliseconds with full risk breakdown.
